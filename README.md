# 뷰리마켓 (Byuri Market)

### 📖 프로젝트 개요
뷰리마켓이라는 중고 물품 거래 웹 플랫폼 입니다. 이 웹은 사용자 등록, 로그인, 물품 등록, 상세 정보 조회, 좋아요 기능 및 삭제 기능을 제공합니다.
<br/><br/>

### 🚀 프로젝트 목표
- 사용자 중고 물품 거래 제공
- 사용자가 물품을 쉽게 등록하고 조회할 수 있는 기능 제공
- 사용자 간 원활한 거래 지원

<br/><br/><br/>

## 📝 프로젝트 설명

### 💼 뷰리마켓의 기능
1. 물품 등록
2. 물품 상세 정보 조회
3. 좋아요 기능
4. 물품 삭제

## ⚙️ 프로젝트 설치 및 실행 방법

### 📝 Prerequisites
- Node.js 및 npm

### 📦 설치 방법
1. 저장소를 클론합니다:
    ```sh
    git clone https://github.com/buseonghyeon/CarrotMarket.git
    ```

2. 필요한 Node.js 패키지를 설치합니다:
    ```sh
    npm install
    ```

### 실행

1. 프로젝트를 실행합니다:
    ```sh
    npm start
    ```
